# Changelog

All notable changes to CSLearn (GCSE Computer Science AI Tutor) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of CSLearn AI tutoring platform
- Student dashboard with topic browsing and question answering
- Teacher dashboard with student progress tracking
- School/org management system with join codes
- AI-powered question generation via OpenRouter
- Supabase authentication with role-based access control
- Dark mode support with theme toggle
- Responsive design for desktop and mobile
- Error boundaries and loading states throughout

### Security
- Row Level Security on all database tables
- Rate limiting on login attempts
- Server-side validation on all API routes
- Password strength requirements enforced
